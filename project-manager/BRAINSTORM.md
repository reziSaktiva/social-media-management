# BRAINSTORM

Dokumen ini menyimpan ide-ide yang muncul dari sesi brainstorming, termasuk ide yang belum matang, alternatif yang dipertimbangkan, dan kemungkinan yang ingin dijelajahi lebih lanjut.

> Diisi otomatis oleh AI setiap kali ada sesi brainstorming yang menghasilkan ide layak simpan.

Ide di sini **belum tentu menjadi keputusan**. Untuk menjadi keputusan, ide harus melalui proses review dan dicatat di `DECISIONS.md`.

---

# Format Entry

```
## [YYYY-MM-DD] — [Topik Brainstorm]
**Phase:** [phase aktif saat brainstorm]
**Context:** [mengapa topik ini didiskusikan]
**Ideas:**
- [ide 1]
- [ide 2]
**Evaluation:** [selaras / bertentangan / perlu ADR / perlu validasi lebih lanjut]
**Next Action:** [discard / simpan untuk fase X / buat ADR / validasi dulu]
```

---

*Belum ada entri. Entri pertama akan ditambahkan saat sesi brainstorming pertama.*
