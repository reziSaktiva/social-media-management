v1.0.0 · 17 July 2026
Audience: Product / UX / UI / Design System
Detail lengkap: Design Brief PDF

Platform web untuk siklus konten marketing: Draft → Schedule → Publish → Engage → Analyze dalam satu dashboard. Analogi: alternatif modern Buffer — sederhana, cepat, dapat dipercaya. Integrasi sosial via Outstand (desain fokus ke connected accounts & status publish, bukan native API).

Draft→ Schedule→ Publish→ Engage→ Analyze

Primary nav (urutan wajib)

1 Home 2 Publish 3 Engage 4 Analyze 5 Start Page

Publish default = Calendar. Badge komentar unread hanya di Engage dan diperbarui setelah sync.

AI hanya inline di Draft Editor — tidak ada menu AI.

Dikunci vs terbuka

Dikunci — jangan diubah tanpa diskusi
  • Urutan primary nav & siklus konten
  • Dua mode Maya/Raka dalam satu UI
  • 6 status + treatment Failed
  • AI hanya inline di editor
  • Content Format per akun (IG/FB: Post·Reel·Story; Pinterest: Pin; TikTok & lainnya: Post) — ADR-039
  • Tidak ada Approval Workflow di MVP
  • UI copy: Bahasa Indonesia
Terbuka — eksplorasi designer (DS-O01–O07)
  • Brand / wordmark & warna primer
  • Type pairing (display + UI sans)
  • Light-only MVP vs dark (preferensi: light)
  • Illustration empty state
  • Kekuatan warna platform di Calendar
  • Detail komponen, motion, polish visual

Filter keputusan visual (ringkas)

Clarity > decoration Trust terlihat (akun, jadwal, Failed) Status sebelum chrome UI Satu pekerjaan per view AI tenang sampai dibutuhkan
